== Description ==
### 3D Menu Awesome - Stunning 3D Off Canvas Menu WordPress Plugin

**3D Menu Awesome** is an awesome [WordPress 3D Menu Plugin](https://themesawesome.com/wordpress-plugin/menu-plugin-wordpress/) that helps You to create the a WordPress menu interface element into your WordPress Site with custom layouts and effects. 3D Menu Awesome is easy to use with extensive choice of 3d layout styles.
